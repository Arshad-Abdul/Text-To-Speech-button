Text-to-Speech Hover Button

Reusable Code

🎤 Live Demo

Try it now! Click the blue speaker button (top right) to activate text-to-speech mode. Then hover your mouse over any text on this page to hear it read aloud.

Sample text to test:

Welcome to the text-to-speech demonstration. This technology helps make websites more accessible.

Hover over this paragraph to hear it spoken aloud. Move your mouse to different elements to hear different content.

Click here to jump to instructions

Features

  • Hover-to-speak functionality for all text elements
  • Beautiful animated button with active state indication
  • Support for reading image alt text for accessibility
  • Clean on/off toggle mechanism
  • Works with Web Speech API (built into modern browsers)
  • No external dependencies required (except Font Awesome for icon)

📋 How to Use This Code on Your Website

  1. Copy the HTML - Paste the button code before your closing </body> tag
  2. Copy the CSS - Add the styles to your stylesheet or in a <style> tag
  3. Copy the JavaScript - Add the script at the bottom of your page before </body>
  4. Include Font Awesome - Add Font Awesome CDN link in your <head> section
Note: This code uses the Web Speech API which is supported in most modern browsers (Chrome, Edge, Safari, Opera). Firefox has partial support.

Step 4: JavaScript (Add before </body>) 

<script>
  // Speaker Button - Toggle Text-to-Speech on Hover Mode
  let speakModeActive = false;
  const speakerBtn = document.getElementById('speakerBtn');
  
  // Click button to toggle speak mode
  speakerBtn.addEventListener('click', function() {
    speakModeActive = !speakModeActive;
    
    if (speakModeActive) {
      this.classList.add('active');
      this.title = 'Speak mode ON - Hover over elements to hear them';
    } else {
      this.classList.remove('active');
      this.title = 'Click to enable speak mode';
      speechSynthesis.cancel();
    }
  });
  
  // Add hover listeners to all elements on the page
  document.addEventListener('mouseover', function(e) {
    if (speakModeActive && e.target !== speakerBtn && !e.target.closest('.speaker-button')) {
      let textContent = '';
      
      // Check if it's an image and has alt text
      if (e.target.tagName === 'IMG' && e.target.alt) {
        textContent = e.target.alt;
      } else {
        // Otherwise get text content
        textContent = e.target.innerText || e.target.textContent;
      }
      
      if (textContent && textContent.trim().length > 0) {
        speechSynthesis.cancel();
        
        const utterance = new SpeechSynthesisUtterance(textContent.trim());
        utterance.rate = 1.0;
        utterance.pitch = 1.0;
        utterance.volume = 1.0;
        
        speechSynthesis.speak(utterance);
      }
    }
  });
</script>

Customization Tips:
  • Change top: 80px; to adjust vertical position
  • Change right: 20px; to adjust horizontal position
  • Modify colors in the gradient to match your site theme
  • Adjust utterance.rate (0.1-10) for speech speed
  • Adjust utterance.pitch (0-2) for voice pitch
